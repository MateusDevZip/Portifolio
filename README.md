## Sobre

Portfólio pessoal desenvolvido com foco em performance, animações suaves e design moderno. Showcasing 3 projetos principais:

- **E-commerce de Alta Performance** — Plataforma B2B/B2C com Next.js e Stripe
- **Sistema ERP** — Dashboard robusto com Vue.js e Django
- **Streaming Social** — App multiplataforma com React Native

---

## Stack Técnico

**Frontend:**
- Vite (build tool)
- Tailwind CSS
- GSAP + ScrollTrigger (animações)
- Lenis.js (smooth scroll)
- Three.js (background 3D)
- Custom Cursor com efeitos

**DevOps:**
- GitHub Pages (hosting)
- GitHub Actions (CI/CD)
- Node.js 20+

---

## Como Rodar Localmente

```bash
# Instalar dependências
npm install

# Servidor de desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview do build
npm run preview
```

Acesse `http://localhost:5173/` (ou a porta indicada no terminal).
